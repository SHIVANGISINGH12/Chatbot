# Chatbot Project

A simple React chatbot demo built with Vite.

## Summary

This is a lightweight chat UI built with React and Vite. It includes components for chat input, messages, and message list. The project demonstrates a minimal frontend for integrating a chatbot UI and is ready for local development and production builds.

## Features

- Minimal React components for chat input and messages
- Vite for fast development and production builds

## Project Structure

- `src/` — application source
  - `components/` — `ChatInput`, `ChatMessage`, `ChatMessages`
  - `assets/` — images
  - `App.jsx`, `main.jsx` — app entry and root component

## Requirements

- Node.js 16+ (or compatible)
- npm or Yarn

## Install

Install dependencies:

```bash
npm install
# or
# yarn
```

## Development

Start the development server:

```bash
npm run dev
```

Open the app at the address Vite reports (typically http://localhost:5173).

## Build

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Customization

- Update or extend components in `src/components/` to change UI or behavior.
- Hook a backend or an API in `ChatInput` to send messages to a chatbot model or service.
